类型 : POST

参数 字段
用户id userid
手机号 mobile 
    返回数据

        {
            "code": "succ",
            "message": "用户信息：",
            "data": {
                "s_userid": "138",      //用户id

        }

搜搜联系人

URL : contact/searchcontact
类型 : POST

参数 字段
用户id userid
搜索类型（1：首字母；2：手机号或姓名） type
搜索内容 name
页数 page 
    返回数据

        {
            "code": "succ",
            "message": "搜索结果:",
            "data": {
                "result": [
                    {
                        "s_username": "刘帅",             //联系人姓名
                        "s_cellphone": "18610792737",       //手机号
                        "s_workunti": "昂",              //工作单位
                        "touserid": "130"               //用户id
                    }
                ],
                "count": "1", //总数
                "flag": 1     //1：是联系人 ；2：不是联系人，但是注册用户；3：不是注册用户
            }
        }
