邮箱状态

URL : login/mailstate
类型 : POST

参数 字段
邮箱 mail 
        返回数据

            {
                code: "succ",
                message: "信息",
                data: [],
                status:""//状态见下边
            }
            1000：缺少参数
            ----------------------------------
            1001：邮箱格式错误
            ----------------------------------
            1002：未注册,点击邀请链接进入，返回公司信息
            ----------------------------------
            1003：链接无效
            ----------------------------------
            1004：未注册，未被邀请
            ----------------------------------
            1005：注册过，返回公司信息
            ----------------------------------
            1006：在邀请名单中,返回公司信息
            ----------------------------------

创建企业

URL : company/createcompany
类型 : POST

参数 字段
邮箱 mail
用户名 username
密码 password
公司名称 companyname
二级域名 domain
城市 city
手机号 mobile
部门 department 
        返回数据

            {
                code: "succ",
                message: "创建成功",
                data: {
                    "userid": "p4YHwoNbT0",    //用户id
                    "companyid": "vDefE3oFip"       //企业id
                },
                status:""
            }
            ----------------------------------
            {
                code: "err",
                message: "创建失败",
                data: [],
                status:""
            }
            1050：此邮箱已被注册
            ----------------------------------
            1051：在邀请名单中呢
            ----------------------------------
            1052：手机号格式不对
            ----------------------------------
            1053：二级域名被占用
            ----------------------------------
            1055：创建失败
